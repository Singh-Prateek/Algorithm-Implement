# Common terms used in array problems
    [2,-1,3,4]

## subarray 
continous and ordered sub elemens

    [3,4] ; [-1,3,4]

## subsequence
**not** continous but ordered sub elements 

    [3,4]

## subset
no order  

    [2]; [-1]; [2,-1] etc.

## algorithm

### Kadane
this gives maxsum in a subarray of an given array. in O(n) time and O(1) space. with slight modification it can get start and end index of the sub array.
